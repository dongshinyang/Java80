#04
- 값 객체(Value Object)를 이용하여 입력 받은 회원정보를 저장하라.  
  동작은 버전 03과 같다.
- 사용 문법:
  => Value Object(사용자 정의 데이터 타입) 클래스의 선언.
     예) Member 클래스  

#05
- 다섯 명의 회원 정보를 입력 받아 출력하라
- 사용 문법:
  => 레퍼런스 배열을 이용하여 여러 인스턴스의 주소를 저장
  => 반복문 활용
